### <a name="depreciation-period"></a><span data-ttu-id="65257-124">Poistojakso</span><span class="sxs-lookup"><span data-stu-id="65257-124">Depreciation Period</span></span>

<span data-ttu-id="65257-125">Jos annat poistojakson (poistovuosien tai poistokuukausien lukumäärän tai poiston lopetuspäivämäärän), poistosumma lasketaan seuraavalla kaavalla:</span><span class="sxs-lookup"><span data-stu-id="65257-125">If you enter the depreciation period (the number of depreciation years, the number of depreciation months, or the depreciation ending date), the following formula calculates the depreciation amount:</span></span>  

<span data-ttu-id="65257-126">*Poistosumma = ((Kirjanpitoarvo - Jäännösarvo) x Poistopäivien lukumäärä) / Jäljellä olevat poistopäivät*</span><span class="sxs-lookup"><span data-stu-id="65257-126">*Depreciation Amount = ((Book value - Salvage Value) x Number of Depreciation Days) / Remaining Depreciation Days*</span></span>

<span data-ttu-id="65257-128">Kirjanpitoarvoa voidaan vähentää kirjatulla arvonkorotuksella, arvonalennuksella, mukautettu 1- tai mukautettu 2 -summilla riippuen siitä, onko **Sisällytä poistolaskentaan** -kentästä poistettu aktivointi ja onko **Kirjanpitoarvon osa** -kenttä aktivoitu **KO:n kirjaustyypin asetukset** -sivulla.</span><span class="sxs-lookup"><span data-stu-id="65257-128">Book value may be reduced by posted appreciation, write-down, custom 1 or custom 2 amounts, depending on whether the **Include in Depr. ### <a name="fixed-yearly-percentage"></a><span data-ttu-id="65257-130">Kiinteä vuosittainen prosentti</span><span class="sxs-lookup"><span data-stu-id="65257-130">Fixed Yearly Percentage</span></span>

### <a name="fixed-yearly-amount"></a><span data-ttu-id="65257-133">Kiinteä vuosittainen summa</span><span class="sxs-lookup"><span data-stu-id="65257-133">Fixed Yearly Amount</span></span>
