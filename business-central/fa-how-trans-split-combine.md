    <span data-ttu-id="c3e03-116">Rivit luodaan nyt käyttöomaisuuden KP-päiväkirjaan käyttämällä mallia ja erää, jotka olet määrittänyt **KO-päiväkirjan asetukset** -sivulla määritellyn poistokirjan osalta.</span><span class="sxs-lookup"><span data-stu-id="c3e03-116">Two lines are now created in the fixed asset G/L journal using the template and batch that you have specified on the **FA Journal Setup** page for the specified depreciation book.</span></span> <span data-ttu-id="c3e03-117">Lisätietoja on kohdassa [Käyttöomaisuuden poiston määrittäminen](fa-how-setup-depreciation.md).</span><span class="sxs-lookup"><span data-stu-id="c3e03-117">For more information, see [Set Up Fixed Asset Depreciation](fa-how-setup-depreciation.md).</span></span>
