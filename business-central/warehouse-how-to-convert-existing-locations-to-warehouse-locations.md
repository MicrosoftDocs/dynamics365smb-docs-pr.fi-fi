## <a name="to-enable-an-existing-location-to-operate-as-a-warehouse-location"></a><span data-ttu-id="1702e-111">Aiemmin luodun sijainnin valmisteleminen fyysisen varastoinnin sijainniksi</span><span class="sxs-lookup"><span data-stu-id="1702e-111">To enable an existing location to operate as a warehouse location</span></span>  
1.  <span data-ttu-id="1702e-112">Valitse ![Lamppu, joka avaa Kerro, mitä haluat tehdä -toiminnon](media/ui-search/search_small.png "Kerro, mitä haluat tehdä") kuvakkeen, syötä **Luo fyys. var. sijainti** ja valitse sitten liittyvä linkki.</span><span class="sxs-lookup"><span data-stu-id="1702e-112">Choose the ![Lightbulb that opens the Tell Me feature](media/ui-search/search_small.png "Tell me what you want to do") icon, enter **Create Warehouse Location**, and then choose the related link.</span></span>  
2.  <span data-ttu-id="1702e-113">Määritä **Sijaintikoodi**-kentässä sijainti, jossa haluat ottaa käyttöön fyysisen varaston käsittelyn.</span><span class="sxs-lookup"><span data-stu-id="1702e-113">In the **Location Code** field, specify the location that you want to enable for warehouse processing.</span></span>  
3.  <span data-ttu-id="1702e-114">Määritä **Muutosvarastopaikan koodi** -kentässä varastopaikka sijainnissa, johon synkronoimattomat fyysisen varastoinnin tapahtumat tallennetaan.</span><span class="sxs-lookup"><span data-stu-id="1702e-114">In the **Adjustment Bin Code** field, specify the bin at the location where unsynchronized warehouse entries are stored.</span></span> <span data-ttu-id="1702e-115">Lisätietoja on kohdassa [Muutettujen fyysisen varastoinnin tapahtumien ja liittyvien nimiketapahtumien synkronointi](inventory-how-count-adjust-reclassify.md#to-synchronize-the-adjusted-warehouse-entries-with-the-related-item-ledger-entries).</span><span class="sxs-lookup"><span data-stu-id="1702e-115">For more information, see the [To synchronize the adjusted warehouse entries with the related item ledger entries](inventory-how-count-adjust-reclassify.md#to-synchronize-the-adjusted-warehouse-entries-with-the-related-item-ledger-entries).</span></span>  

    <span data-ttu-id="1702e-116">Käyttämällä määritetylle sijainnille avoimia nimiketapahtumia ohjelma luo fyysisen varastoinnin päiväkirjarivit, joilla on kaikki mahdolliset nimiketapahtumissa olevat Nimikkeen nro-, Varianttikoodi-, Mittayksikön koodi- ja tarvittaessa Eränro- ja Sarjanro-kenttien yhdistelmät.</span><span class="sxs-lookup"><span data-stu-id="1702e-116">Using the open item ledger entries for the specified location, warehouse journal lines are created that sum up every combination of Item No., Variant Code, Unit of Measure Code, and, if necessary, Lot No. and Serial No. in the item ledger entries.</span></span> <span data-ttu-id="1702e-117">Ohjelma kirjaa sen jälkeen fyysisen varastoinnin päiväkirjarivit.</span><span class="sxs-lookup"><span data-stu-id="1702e-117">The warehouse journal lines are then posted.</span></span> <span data-ttu-id="1702e-118">Tämä kirjaus luo fyysisen varastoinnin tapahtumat, jotka sijoittavat varaston fyysisen varastoinnin muutosvarastopaikkaan.</span><span class="sxs-lookup"><span data-stu-id="1702e-118">This posting creates warehouse entries that place the inventory in the warehouse adjustment bin.</span></span> <span data-ttu-id="1702e-119">Ohjelma määrittää myös sijaintikortin **Muutosvarastopaikan koodi**-kentän arvon.</span><span class="sxs-lookup"><span data-stu-id="1702e-119">The **Adjustment Bin Code** on the location card is also set.</span></span>  

4.  <span data-ttu-id="1702e-120">Voit tarkastella muutosvarastopaikkaan eräajon aikana lisättyjä nimikkeitä suorittamalla **F. var. muutosvar.paikka** -raportin.</span><span class="sxs-lookup"><span data-stu-id="1702e-120">To see which items were added to the adjustment bin during the batch job, run the **Warehouse Adjustment Bin** report.</span></span>  
5.  <span data-ttu-id="1702e-121">Kun **Luo fyys. var. sijainti** -eräajo on valmis, suorita ja kirjaa fyysisen varastoinnin inventointi.</span><span class="sxs-lookup"><span data-stu-id="1702e-121">When the **Create Warehouse Location** batch job has completed, perform and post a warehouse physical inventory.</span></span> <span data-ttu-id="1702e-122">Lisätietoja on kohdassa [Varaston laskeminen, muuttaminen ja uudelleenluokitus päiväkirjojen avulla](inventory-how-count-adjust-reclassify.md).</span><span class="sxs-lookup"><span data-stu-id="1702e-122">For more information, see [Count, Adjust, and Reclassify Inventory Using Journals](inventory-how-count-adjust-reclassify.md).</span></span>  

> [!NOTE]  
>  <span data-ttu-id="1702e-123">On suositeltavaa suorittaa **Luo fyys. var. sijainti** -eräajo sellaiseen aikaan, ettei se häiritse järjestelmän tavallista käyttöä.</span><span class="sxs-lookup"><span data-stu-id="1702e-123">It is recommended that you run the **Create Warehouse Location** batch job at a time when it will not impact the daily work in the system.</span></span> <span data-ttu-id="1702e-124">Tämä eräajo käsittelee jokaisen **Nimiketapahtuma**-taulukon tapahtuman, ja jos nimiketapahtumia on paljon, eräajo voi kestää useita tunteja.</span><span class="sxs-lookup"><span data-stu-id="1702e-124">This job processes each entry in the **Item Ledger Entry** table, and if there are a large number of item ledger entries, the job can last several hours.</span></span>  

 <span data-ttu-id="1702e-125">Sijainneissa, joissa ei ole käytetty varastoinninhallinnan asiakirjoja ennen muunnosta, kaikki ennen muunnosta vastaanotetut tai osittain toimitetut lähdeasiakirjat on avattava uudelleen ja vapautettava.</span><span class="sxs-lookup"><span data-stu-id="1702e-125">For those locations that did not use warehouse management documents before the conversion, you must re-open and release any source documents that were partially received or partially shipped before the conversion.</span></span>
