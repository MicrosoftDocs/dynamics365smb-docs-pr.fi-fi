# <a name="work-with-manufacturing-batch-units-of-measure"></a><span data-ttu-id="7d121-104">Tuotantoerän mittayksiköiden käyttäminen</span><span class="sxs-lookup"><span data-stu-id="7d121-104">Work with Manufacturing Batch Units of Measure</span></span>
<span data-ttu-id="7d121-105">Jos nimike varastoidaan yhtä mittayksikköä ja tuotetaan toista mittayksikköä käyttäen, ohjelma voi laskea komponenttien oikean määrän **Päivitä tuotantotilaus** -eräajon aikana luomalla tuotantoerän mittayksikköä käyttävän tuotantotilauksen.</span><span class="sxs-lookup"><span data-stu-id="7d121-105">If an item is stocked in one unit of measure but produced in another, a production order is created that uses a manufacturing batch unit of measure to calculate the correct quantity of the components during the **Refresh Production Order** batch job.</span></span> <span data-ttu-id="7d121-106">Tuotantoerän mittayksikön laskentaa voidaan tarvita esimerkiksi, kun nimike varastoidaan kappaleina mutta tuotetaan tonneina.</span><span class="sxs-lookup"><span data-stu-id="7d121-106">An example of a manufacturing batch unit of measure calculation is when a manufactured item is stocked in pieces but produced in tons.</span></span>  

## <a name="to-create-a-production-bom-using-a-batch-unit-of-measure"></a><span data-ttu-id="7d121-107">Tuotannon tuoterakenteen luominen erän mittayksikköä käyttäen</span><span class="sxs-lookup"><span data-stu-id="7d121-107">To create a production BOM using a batch unit of measure</span></span>  
1.  <span data-ttu-id="7d121-108">Tuotantoerän mittayksikkö määritetään vaihtoehtoiseksi mittayksiköksi tuotettavan nimikkeen **Nimikkeen mittayksiköt** -sivulla.</span><span class="sxs-lookup"><span data-stu-id="7d121-108">The manufacturing batch unit of measure is set up as an alternative unit of measure on the **Item Units of Measure** page on the item to be produced.</span></span> <span data-ttu-id="7d121-109">Erän mittayksikkö ei korvaa nimikkeen perusmittayksikköä.</span><span class="sxs-lookup"><span data-stu-id="7d121-109">The batch unit of measure will not replace the base unit of measure on the item.</span></span>  
2.  <span data-ttu-id="7d121-110">Luo tuotannon tuoterakenne nimikkeelle, johon on määritetty tuotantoerän mittayksikkö.</span><span class="sxs-lookup"><span data-stu-id="7d121-110">Create a production BOM for the item set up with the manufacturing batch unit of measure.</span></span> <span data-ttu-id="7d121-111">Lisätietoja on kohdassa [Tuotannon tuoterakenteiden luominen](production-how-to-create-production-boms.md).</span><span class="sxs-lookup"><span data-stu-id="7d121-111">For more information, see [Create Production BOMs](production-how-to-create-production-boms.md).</span></span>  
3.  <span data-ttu-id="7d121-112">Napsauta **Mittayksikön koodi** -kenttää ja valitse tuotantoerän mittayksikön koodi.</span><span class="sxs-lookup"><span data-stu-id="7d121-112">In the **Unit of Measure Code** field, select the manufacturing batch unit of measure.</span></span>  
4.  <span data-ttu-id="7d121-113">Lisää tuotannon tuoterakenteen jokaiselle riville **Määrä per** -kenttään tämän komponenttinimikkeen määrä, joka tarvitaan tämän erän mittayksikön luomiseen.</span><span class="sxs-lookup"><span data-stu-id="7d121-113">For each production BOM line, in the **Quantity Per** field, enter the quantity of this component item that is required to create this batch unit of measure.</span></span>  
5.  <span data-ttu-id="7d121-114">Avaa liittyvän nimikkeen **Nimikkeen kortti**.</span><span class="sxs-lookup"><span data-stu-id="7d121-114">Open the **Item Card** for the related item.</span></span>  

    <span data-ttu-id="7d121-115">Valitse **Täydennys**-pikavälilehden **Tuotannon tuoterakenteen nro** -kentässä edellä luotu tuotannon tuoterakenne.</span><span class="sxs-lookup"><span data-stu-id="7d121-115">On the **Replenishment** FastTab, in the **Production BOM No.** field, select the production BOM created above.</span></span>  
6.  <span data-ttu-id="7d121-116">Luo tuotantotilauksen otsikko käyttäen nimikettä, johon on määritetty tuotantoerän mittayksikkö.</span><span class="sxs-lookup"><span data-stu-id="7d121-116">Create a production order header using the item set up with the manufacturing batch unit of measure.</span></span> <span data-ttu-id="7d121-117">Lisätietoja on kohdassa [Tuotantotilausten luominen](production-how-to-create-production-orders.md).</span><span class="sxs-lookup"><span data-stu-id="7d121-117">For more information, see [Create Production Orders](production-how-to-create-production-orders.md).</span></span>  
7.  <span data-ttu-id="7d121-118">Valitse ensin **Päivitä**-toiminto ja sitten **OK**.</span><span class="sxs-lookup"><span data-stu-id="7d121-118">Choose the **Refresh** action, and then choose  the **OK** button.</span></span>  

<span data-ttu-id="7d121-119">Voit tarkastella tuloksia valitsemalla ensin **Rivit**-pikavälilehdessä **Rivi**- ja sitten **Komponentit**-toiminnon.</span><span class="sxs-lookup"><span data-stu-id="7d121-119">On the **Lines** FastTab, choose the **Line** action, and then choose the **Components** action to view the result.</span></span> <span data-ttu-id="7d121-120">Sovellus laskee tuotantoerän mittayksikön perusteella oikean komponenttimäärän, jonka tuotannon tuoterakenne tarvitsee.</span><span class="sxs-lookup"><span data-stu-id="7d121-120">The application calculates the correct quantity of the components needed to satisfy the production BOM based on the manufacturing batch unit of measure.</span></span>  

## <a name="to-calculate-a-manufacturing-batch-unit-of-measure-on-a-production-order"></a><span data-ttu-id="7d121-121">Tuotantotilauksen tuotantoerän mittayksikön laskeminen</span><span class="sxs-lookup"><span data-stu-id="7d121-121">To calculate a manufacturing batch unit of measure on a production order</span></span>  
1.  <span data-ttu-id="7d121-122">Luo tuotantotilauksen otsikko käyttäen nimikettä, johon on määritetty tuotantoerän mittayksikkö.</span><span class="sxs-lookup"><span data-stu-id="7d121-122">Create a production order header using the item set up with the manufacturing batch unit of measure.</span></span>  
2.  <span data-ttu-id="7d121-123">Kirjoita otsikossa käytetty nimikkeen numero tuotantotilauksen rivin **Nimikkeen nro** -kenttään.</span><span class="sxs-lookup"><span data-stu-id="7d121-123">In the **Item No.** field in the Production Order line, type the same item number used in the header.</span></span>  
3.  <span data-ttu-id="7d121-124">Lisää otsikossa käytetty määrä **Määrä**-kenttään.</span><span class="sxs-lookup"><span data-stu-id="7d121-124">In the **Quantity** field, enter the same quantity used in the header.</span></span>  
4.  <span data-ttu-id="7d121-125">Napsauta **Mittayksikön koodi** -kenttää ja valitse tuotantoerän mittayksikön koodi.</span><span class="sxs-lookup"><span data-stu-id="7d121-125">In the **Unit of Measure Code** field, select the manufacturing batch unit of measure code.</span></span>  
5.  <span data-ttu-id="7d121-126">Valitse **Päivitä**-toiminto.</span><span class="sxs-lookup"><span data-stu-id="7d121-126">Choose the **Refresh** action.</span></span>
6.  <span data-ttu-id="7d121-127">Poista **Laske**-pikavälilehden **Rivit**-valintaruudun valinta.</span><span class="sxs-lookup"><span data-stu-id="7d121-127">On the **Calculate** FastTab, clear the **Lines** check box.</span></span>  
7.  <span data-ttu-id="7d121-128">Valitse **OK**-painike.</span><span class="sxs-lookup"><span data-stu-id="7d121-128">Choose the **OK** button.</span></span>  
8.  <span data-ttu-id="7d121-129">Voit tarkastella tuloksia valitsemalla ensin **Rivit**-pikavälilehdessä **Rivi**- ja sitten **Komponentit**-toiminnon.</span><span class="sxs-lookup"><span data-stu-id="7d121-129">On the **Lines** FastTab, choose the **Line** action, and then choose the **Components** action to view the result.</span></span> <span data-ttu-id="7d121-130">Ohjelma laskee tuotantoerän mittayksikön perusteella oikean komponenttimäärän, jonka tuotannon tuoterakenne tarvitsee.</span><span class="sxs-lookup"><span data-stu-id="7d121-130">The correct quantity of the components needed to satisfy the production BOM is calculated based on the manufacturing batch unit of measure.</span></span>
