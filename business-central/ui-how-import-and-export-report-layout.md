<span data-ttu-id="b2c28-104">Voit tuoda ja viedä olemassa olevan mukautetun raporttiasettelun tiedostona tietokoneen tai verkon sijainnista ja sijaintiin.</span><span class="sxs-lookup"><span data-stu-id="b2c28-104">You can import and export an existing custom report layout as a file to and from a location on your computer and network.</span></span> <span data-ttu-id="b2c28-105">Voit esimerkiksi viedä raporttiasettelun ja lähettää sitten tiedoston toiselle henkilölle muokkausta varten.</span><span class="sxs-lookup"><span data-stu-id="b2c28-105">For example, you can export a report layout, and then send the file to another person to modify.</span></span> <span data-ttu-id="b2c28-106">Kyseinen henkilö voi tehdä muutoksia asetteluun ja palauttaa tiedoston sinulle, jotta voit tuoda sen takaisin.</span><span class="sxs-lookup"><span data-stu-id="b2c28-106">That person can then make the modifications to layout and return the file to you so that you can import it back.</span></span>

>  <span data-ttu-id="b2c28-107">Et voi tuoda tai viedä valmiita raporttiasetteluita.</span><span class="sxs-lookup"><span data-stu-id="b2c28-107">You cannot import or export built-in report layouts.</span></span>  

### <a name="to-export-a-report-layout-to-a-file"></a><span data-ttu-id="b2c28-108">Raporttiasettelun vienti tiedostoon</span><span class="sxs-lookup"><span data-stu-id="b2c28-108">To export a report layout to a file</span></span>  

1.  <span data-ttu-id="b2c28-109">Valitse ![Lamppu, joka avaa Kerro, mitä haluat tehdä -toiminnon](media/ui-search/search_small.png "Kerro, mitä haluat tehdä") kuvakkeen, syötä **Raporttiasetteluvalinta** ja valitse sitten liittyvä linkki.</span><span class="sxs-lookup"><span data-stu-id="b2c28-109">Choose the ![Lightbulb that opens the Tell Me feature](media/ui-search/search_small.png "Tell me what you want to do") icon, enter **Report Layout Selection**, and then choose the related link.</span></span>  

2.  <span data-ttu-id="b2c28-110">Valitse raporttirivi, joka sisältää mukautetun raporttiasettelun, jonka haluat viedä, ja valitse sitten **Mukautetut asettelut** -toiminto.</span><span class="sxs-lookup"><span data-stu-id="b2c28-110">Select the row for the report that contains the custom report layout that you want to export, and then choose the **Custom Layouts** action.</span></span>  

3.  <span data-ttu-id="b2c28-111">Valitse **Raporttiasettelut** -sivulla raporttiasettelu, jonka haluat viedä tiedostoon ja valitse sitten **Vie asettelu** -toiminto.</span><span class="sxs-lookup"><span data-stu-id="b2c28-111">On the **Report Layouts** page, select the report layout that you want to export to a file, and then choose the **Export Layout** action.</span></span>  

4.  <span data-ttu-id="b2c28-112">Valitse **Vie tiedosto** -valintaikkunassa **Tallenna**-painike ja tallenna tiedosto sitten uuteen paikkaan tietokoneelle tai verkkoon.</span><span class="sxs-lookup"><span data-stu-id="b2c28-112">In the **Export File** dialog box, choose the **Save** button, and then save the file to a location on your computer or network.</span></span>  

### <a name="to-import-a-report-layout-file"></a><span data-ttu-id="b2c28-113">Raporttiasettelun tiedoston tuonti</span><span class="sxs-lookup"><span data-stu-id="b2c28-113">To import a report layout file</span></span>  

1.  <span data-ttu-id="b2c28-114">Varmista, että raporttiasettelun määrittävä asianmukainen tiedosto on käytettävissä tietokoneellasi tai verkossa.</span><span class="sxs-lookup"><span data-stu-id="b2c28-114">Make sure that the relevant file that defines the report layout is available on your computer or network.</span></span>  

     <span data-ttu-id="b2c28-115">Word-raporttiasettelutiedoston tiedostotyypin on oltava .docx.</span><span class="sxs-lookup"><span data-stu-id="b2c28-115">A Word report layout file must have the .docx file type extension.</span></span> <span data-ttu-id="b2c28-116">RDLC-raporttiasettelutiedoston on oltava .rdlc- tai RDL-tiedostotyyppiä.</span><span class="sxs-lookup"><span data-stu-id="b2c28-116">An RDLC report layout file must have the .rdlc or .rdl file type extension.</span></span>  

2.  <span data-ttu-id="b2c28-117">Valitse ![Lamppu, joka avaa Kerro, mitä haluat tehdä -toiminnon](media/ui-search/search_small.png "Kerro, mitä haluat tehdä") kuvakkeen, syötä **Raporttiasetteluvalinta** ja valitse sitten liittyvä linkki.</span><span class="sxs-lookup"><span data-stu-id="b2c28-117">Choose the ![Lightbulb that opens the Tell Me feature](media/ui-search/search_small.png "Tell me what you want to do") icon, enter **Report Layout Selection**, and then choose the related link.</span></span>  

3.  <span data-ttu-id="b2c28-118">Valitse sen raportin rivi, johon haluat tuoda raporttiasettelun, ja valitse sitten **Mukautetut asettelut** -toiminto.</span><span class="sxs-lookup"><span data-stu-id="b2c28-118">Select the row for the report to which you want to import the report layout, and then choose the **Custom Layouts** action.</span></span>  

4.  <span data-ttu-id="b2c28-119">Valitse **Raporttiasettelut** -sivulla raporttiasettelu, johon haluat tuoda tiedoston ja valitse sitten **Tuo asettelu** -toiminto.</span><span class="sxs-lookup"><span data-stu-id="b2c28-119">On the **Report Layouts** page, select the report layout to which you want to import the file, and then choose the **Import Layout** action.</span></span>  

5.  <span data-ttu-id="b2c28-120">Valitse **Tuo**-valintaruudussa asiakirja, joka määrittää raporttiasettelun ja valitse sitten **Avaa**-painike.</span><span class="sxs-lookup"><span data-stu-id="b2c28-120">In the **Import** dialog box, select the document that defines the report layout, and then choose the **Open** button.</span></span>  

 <span data-ttu-id="b2c28-121">Alkuperäinen mukautettu raporttiasettelu korvataan tuodulla raporttiasettelulla.</span><span class="sxs-lookup"><span data-stu-id="b2c28-121">The original custom report layout is replaced with the imported report layout.</span></span>
