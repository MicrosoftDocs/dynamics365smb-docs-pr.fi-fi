    <span data-ttu-id="efa41-108">**Rekisteriin nro** -kentässä</span><span class="sxs-lookup"><span data-stu-id="efa41-108">The **To Register No.**</span></span> <span data-ttu-id="efa41-109">on viimeisen rekisteritapahtuman numero, eikä sitä voi muuttaa.</span><span class="sxs-lookup"><span data-stu-id="efa41-109">field contains the last register entry number and cannot be changed.</span></span>  

    <span data-ttu-id="efa41-110">Voit käyttää **Rekisteristä nro** -kenttää</span><span class="sxs-lookup"><span data-stu-id="efa41-110">You can use the **From Register No.**</span></span> <span data-ttu-id="efa41-111">valitaksesi rekisteritapahtuman numeron, josta poisto pitäisi alkaa.</span><span class="sxs-lookup"><span data-stu-id="efa41-111">field to select a register entry number from which the deletion should begin.</span></span>
