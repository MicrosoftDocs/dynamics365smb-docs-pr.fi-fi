## <a name="types-of-user-settings"></a><span data-ttu-id="3e5fe-109">Käyttäjäasetusten tyypit</span><span class="sxs-lookup"><span data-stu-id="3e5fe-109">Types of user settings</span></span>

<span data-ttu-id="3e5fe-110">*Käyttäjäasetukset* eivät ole sama asia kuin *käyttäjän määritykset*, jotka koskevat käyttäjää entiteettinä ja käyttäjän järjestelmän käyttöoikeuksia.</span><span class="sxs-lookup"><span data-stu-id="3e5fe-110">*User settings* is not the same as *user setup*, which is about the user as an entity and the user's access in the system.</span></span> <span data-ttu-id="3e5fe-111">Käyttäjäasetuksilla ei ole myöskään mitään tekemistä käyttäjän mukauttamisella, mikä tarkoittaa pinnallisia käyttöliittymän muutoksia.</span><span class="sxs-lookup"><span data-stu-id="3e5fe-111">Furthermore, user settings has nothing to do with a user's personalization, such as lightweight changes to the user interface.</span></span> <span data-ttu-id="3e5fe-112">Käyttäjäasetuksilla määritetään kunkin käyttäjän esimääritetyt asetukset siltä osin, miten sovelluksen eri osat näkyvät käyttäjälle.</span><span class="sxs-lookup"><span data-stu-id="3e5fe-112">User settings determine the predefined settings for each user in various aspects of the way the application presents itself to the user.</span></span> <span data-ttu-id="3e5fe-113">Seuraavaksi käsitellään viisi käyttäjäasetus- ja määritystyyppiä, jotka yksittäinen käyttäjä voi määrittää tai jotka järjestelmänvalvoja voi määrittää keskitetysti.</span><span class="sxs-lookup"><span data-stu-id="3e5fe-113">The following paragraph lists the five types of user settings and preferences that can be set by the individual or centrally by the administrator:</span></span>

- <span data-ttu-id="3e5fe-117">**Rooli**</span><span class="sxs-lookup"><span data-stu-id="3e5fe-117">**Role**</span></span>  

  <span data-ttu-id="3e5fe-118">Rooli tai profiili tarkoittaa käyttäjän tehtävää yrityksessä, kuten *myyntipäällikköä*, *kirjanpitäjää* tai *ostajaa*.</span><span class="sxs-lookup"><span data-stu-id="3e5fe-118">The role, or profile, describes the user's function in the company, such as *Sales Manager*, *Bookkeeper*, or *Purchasing Agent*.</span></span> <span data-ttu-id="3e5fe-119">Profiili määrittää sitten käyttäjän roolikeskuksen eli aloitussivun, joka avautuu käyttäjille kirjauduttaessa.</span><span class="sxs-lookup"><span data-stu-id="3e5fe-119">The profile then determines the user's role center, the home page that users will see when they sign in.</span></span> <span data-ttu-id="3e5fe-120">Profiili ei vaikuta [!INCLUDE[prod_short](includes/prod_short.md)]in toimintojen käyttöoikeuksiin.</span><span class="sxs-lookup"><span data-stu-id="3e5fe-120">The profile does not impact access rights to functionality in [!INCLUDE[prod_short](includes/prod_short.md)].</span></span>

  <span data-ttu-id="3e5fe-126">Jos Microsoft 365:n kieliasetus vastaa [!INCLUDE[prod_short](includes/prod_short.md)]in tukemaa kieltä, kyseinen kieli valitaan käyttäjälle.</span><span class="sxs-lookup"><span data-stu-id="3e5fe-126">If the language setting from Microsoft 365 matches a supported language in [!INCLUDE[prod_short](includes/prod_short.md)], this language will be chosen for the user.</span></span>

  > <span data-ttu-id="3e5fe-127">Kielen näyttäminen oikein voi edellyttää, [!INCLUDE[prod_short](includes/prod_short.md)]in kielisovelluksen asentamista.</span><span class="sxs-lookup"><span data-stu-id="3e5fe-127">You may have to install a language app for [!INCLUDE[prod_short](includes/prod_short.md)] to properly display the language.</span></span> <span data-ttu-id="3e5fe-128">Tämän vuoksi tarvittavat kielisovelluksen kannattaakin asentaa, ennen kuin kukaan on kirjautunut sisään ensimmäisen kerran, jotta käyttökokemus on hyvä alusta alkaen.</span><span class="sxs-lookup"><span data-stu-id="3e5fe-128">Therefore, it is a good practice to install the necessary language apps before any user logs in the first time so that they have a good experience from their first day.</span></span> <span data-ttu-id="3e5fe-129">Lisätietoja on [tuettujen kielien](/dynamics365/business-central/dev-itpro/compliance/apptest-countries-and-translations) luettelossa.</span><span class="sxs-lookup"><span data-stu-id="3e5fe-129">For more information, see the list of [supported languages](/dynamics365/business-central/dev-itpro/compliance/apptest-countries-and-translations).</span></span>  
  
- <span data-ttu-id="3e5fe-130">**Alue**</span><span class="sxs-lookup"><span data-stu-id="3e5fe-130">**Region**</span></span>

- <span data-ttu-id="3e5fe-134">**Aikavyöhyke**</span><span class="sxs-lookup"><span data-stu-id="3e5fe-134">**Time zone**</span></span>  

  <span data-ttu-id="3e5fe-135">Määrittää käyttäjän aikavyöhykkeen.</span><span class="sxs-lookup"><span data-stu-id="3e5fe-135">Defines the time zone in which the user is located.</span></span> <span data-ttu-id="3e5fe-136">Tätä ei tällä hetkellä synkronoida Microsoft 365:stä, joten se on määritettävä manuaalisesti.</span><span class="sxs-lookup"><span data-stu-id="3e5fe-136">Currently this is not synchronized from Microsoft 365 and must be set manually.</span></span>  

- <span data-ttu-id="3e5fe-137">**Opastusvihjeitä**</span><span class="sxs-lookup"><span data-stu-id="3e5fe-137">**Teaching tips**</span></span>
