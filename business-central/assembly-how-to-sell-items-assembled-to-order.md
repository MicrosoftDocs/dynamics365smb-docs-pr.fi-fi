# <a name="sell-items-assembled-to-order"></a><span data-ttu-id="6dcb4-104">Kokoonpano tilausta varten -nimikkeiden myyminen</span><span class="sxs-lookup"><span data-stu-id="6dcb4-104">Sell Items Assembled to Order</span></span>
<span data-ttu-id="6dcb4-105">Jos kokoonpanon nimikkeen kortin **Kokoonpanokäytäntö**-kenttä sisältää **Kokoonpano tilausta varten** -kohdan, nimikettä ei oleteta olevan varastossa, vaan se on koottava myyntitilausta varten.</span><span class="sxs-lookup"><span data-stu-id="6dcb4-105">If the **Assembly Policy** field on the item card of an assembly item is **Assemble-to-Order**, then the item is not expected to be in inventory, and it must be assembled specifically to a sales order.</span></span> <span data-ttu-id="6dcb4-106">Kun syötät nimikkeen myyntitilausriville, kokoonpanotilaus luodaan automaattisesti ja linkitetään myyntitilaukseen.</span><span class="sxs-lookup"><span data-stu-id="6dcb4-106">When you enter the item on a sales order line, then an assembly order is automatically created and linked to the sales order.</span></span>  

> [!NOTE]

<span data-ttu-id="6dcb4-109">Tässä toimenpiteessä käsittelet asiakkaan pyytämien määrittelyjen mukaan koottavan nimikkeen myyntiä.</span><span class="sxs-lookup"><span data-stu-id="6dcb4-109">In this procedure, you process the sale of an item that will be assembled according to specifications that are requested by the customer.</span></span> <span data-ttu-id="6dcb4-110">Vaiheita ovat myyntitilausrivin käynnistäminen, kokoonpanonimikkeen mukauttaminen muokkaamalla sen osia ja resursseja, saatavuuden tarkistaminen toimituspäivämäärän määrittämiseksi ja myyntitilauksen vapauttaminen kokoonpanoa ja välitöntä toimitusta varten.</span><span class="sxs-lookup"><span data-stu-id="6dcb4-110">The steps include initiating the sales order line, customizing the assembly item by editing its components and resources, checking availability to establish a delivery date, and releasing the sales order to be assembled and immediately shipped.</span></span>  

> [!NOTE]  
>  <span data-ttu-id="6dcb4-111">Seuraavassa toimenpiteessä ei tehdä myyntitilauksen vakiovaiheita ennen, kuin koontitilausnimike syötetään myyntitilauksen riville.</span><span class="sxs-lookup"><span data-stu-id="6dcb4-111">The following procedure does not include the standard sales order steps before the step where you enter the assemble-to-order item on a sales order line.</span></span>  

## <a name="to-sell-an-item-that-is-assembled-to-order"></a><span data-ttu-id="6dcb4-112">Myy nimike, jok aon kokoonpano tilausta varten</span><span class="sxs-lookup"><span data-stu-id="6dcb4-112">To sell an item that is assembled to order</span></span>

2.  <span data-ttu-id="6dcb4-114">Luo myyntitilaus.</span><span class="sxs-lookup"><span data-stu-id="6dcb4-114">Create a sales order.</span></span> <span data-ttu-id="6dcb4-115">Lisätietoja on kohdassa [Tuotteiden myyminen](sales-how-sell-products.md).</span><span class="sxs-lookup"><span data-stu-id="6dcb4-115">For more information, see [Sell Products](sales-how-sell-products.md).</span></span>  
3.  <span data-ttu-id="6dcb4-116">Valitse **Nro**-kenttään</span><span class="sxs-lookup"><span data-stu-id="6dcb4-116">In the **No.**</span></span> <span data-ttu-id="6dcb4-117">nimike, jolle on määritetty kokoonpano tilausta varten.</span><span class="sxs-lookup"><span data-stu-id="6dcb4-117">field, enter an item that is set up to be assembled to order.</span></span>

5.  <span data-ttu-id="6dcb4-120">Määritä **Määrä**-kentässä, miten monta yksikköä haluat myytävän.</span><span class="sxs-lookup"><span data-stu-id="6dcb4-120">In the **Quantity** field, enter how many units to sell.</span></span>  

    > [!NOTE]  
    >  <span data-ttu-id="6dcb4-121">Jos vähintään yksi pyydetyn kokoonpanon nimikkeen määrän komponenteista ei ole saatavana, näyttöön avautuu yksityiskohtaisia saatavuustietoja sisältävä varoitussivu.</span><span class="sxs-lookup"><span data-stu-id="6dcb4-121">If one or more components of the requested assembly item quantity are not available, then a detailed availability warning page opens.</span></span> <span data-ttu-id="6dcb4-122">Lisätietoja on kohdassa Kokoonpanon saatavuus.</span><span class="sxs-lookup"><span data-stu-id="6dcb4-122">For more information, see Assembly Availability.</span></span>  

    <span data-ttu-id="6dcb4-123">Kokoonpanotilaus luodaan nyt automaattisesti ja linkitetään myyntitilauksen riviin.</span><span class="sxs-lookup"><span data-stu-id="6dcb4-123">An assembly order is now automatically created and linked to the sales order line.</span></span> <span data-ttu-id="6dcb4-124">Kokoonpanotilauksen eräpäivä synkronoidaan myyntitilausrivin lähetyspäivämäärän kanssa.</span><span class="sxs-lookup"><span data-stu-id="6dcb4-124">The due date of this assembly order is synchronized with the shipment date of the sales order line.</span></span>  

    <span data-ttu-id="6dcb4-125">Myytävä määrä on kopioitu **Kokoonpantava määrä tilausta varten** -kentästä, joka ilmaisee, että nimikkeen asennus vaatii myyntirivillä olevan täyden määrän kokoonpanon tilausta varten.</span><span class="sxs-lookup"><span data-stu-id="6dcb4-125">The quantity to sell is copied to the **Qty. to Assemble to Order** field, which indicates that the item setup expects the full quantity on the sales line to be assembled to the order.</span></span> <span data-ttu-id="6dcb4-126">Voit vähentää kokoonpano tilausta varten -määrää, jos tiedät, että jotkin osat ovat jo saatavilla.</span><span class="sxs-lookup"><span data-stu-id="6dcb4-126">You can decrease the quantity to assemble to order, such as if you know that some items are already available.</span></span> <span data-ttu-id="6dcb4-127">Lisätietoja on kohdassa [Varastonimikkeiden myyminen Kokoonpano tilausta varten -työnkuluissa](assembly-how-to-sell-inventory-items-in-assemble-to-order-flows.md).</span><span class="sxs-lookup"><span data-stu-id="6dcb4-127">For more information, see [Sell Inventory Items in Assemble-to-Order Flows](assembly-how-to-sell-inventory-items-in-assemble-to-order-flows.md).</span></span>

7.  <span data-ttu-id="6dcb4-130">Luo **Kokoonpano tilausta varten -rivit** -sivulla pyydettyä lisäkokoonpanosisältöä varten uusi rivi, jonka tyyppi on **Nimike**.</span><span class="sxs-lookup"><span data-stu-id="6dcb4-130">On the **Assemble-to-Order Lines** page, create a new line of type **Item** for the requested additional kit content.</span></span> <span data-ttu-id="6dcb4-131">Rivi vastaa ylimääräistä kokoonpano-osaa.</span><span class="sxs-lookup"><span data-stu-id="6dcb4-131">The line represents an additional assembly component.</span></span>
