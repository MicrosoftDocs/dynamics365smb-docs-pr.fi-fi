## <a name="to-register-consumption-and-output"></a><span data-ttu-id="014aa-118">Kulutuksen ja tuotoksen rekisteröiminen</span><span class="sxs-lookup"><span data-stu-id="014aa-118">To register consumption and output</span></span>  
1.  <span data-ttu-id="014aa-119">Valitse ![Lamppu, joka avaa Kerro, mitä haluat tehdä -toiminnon](media/ui-search/search_small.png "Kerro, mitä haluat tehdä") -kuvake, syötä **Vapautetut tuotantotilaukset** ja valitse sitten liittyvä linkki.</span><span class="sxs-lookup"><span data-stu-id="014aa-119">Choose the ![Lightbulb that opens the Tell Me feature](media/ui-search/search_small.png "Tell me what you want to do") icon enter **Released Prod. 2.  <span data-ttu-id="014aa-120">Avaa rekisteröimistä odottava vapautettu tuotantotilausrivi ja valitse sitten **Rivit**-pikavälilehdessä ensin **Rivi**-toiminto ja lopuksi **Tuotantopäiväkirja**-toiminto.</span><span class="sxs-lookup"><span data-stu-id="014aa-120">Open a released production order line that is ready for registration, and then on the **Lines** FastTab, choose the **Line** action, and then choose the **Production Journal** action.</span></span>  

    <span data-ttu-id="014aa-121">**Tuotantopäiväkirja**-sivu avautuu, ja siinä näkyvät tuotantotilausrivin arviointiperusteeksi päiväkirjarivit **Tuotantotilauksen komponentti** ja **Tuotantotilauksen reititys** -sivujen mukaisesti.</span><span class="sxs-lookup"><span data-stu-id="014aa-121">The **Production Journal** page opens showing journal lines for the production order line according to the **Prod. 3.  <span data-ttu-id="014aa-124">Syötä päiväkirjan yläreunan **Kirjauspvm**-kenttään kirjauspäivämäärä, jota käytetään kaikilla riveillä.</span><span class="sxs-lookup"><span data-stu-id="014aa-124">In the **Posting Date** field at the top of the journal, enter a posting date that applies to all lines.</span></span> <span data-ttu-id="014aa-125">Oletusarvona on käsittelypäivämäärä.</span><span class="sxs-lookup"><span data-stu-id="014aa-125">The work date is entered by default.</span></span> <span data-ttu-id="014aa-126">Kenttä on tarkoitettu nopeaksi keinoksi asettaa kaikille riveille tarvittaessa yhtenäiset kirjauspäivämäärät.</span><span class="sxs-lookup"><span data-stu-id="014aa-126">The field is meant as a quick way to align posting dates on all lines, if relevant.</span></span>  

    > [!NOTE]  
    >  <span data-ttu-id="014aa-127">Itse riville annettu kirjauspäivämäärä kumoaa tämän kentän arvon.</span><span class="sxs-lookup"><span data-stu-id="014aa-127">Posting dates entered on individual lines will override this field.</span></span>  

4.  <span data-ttu-id="014aa-128">Päiväkirjan yläreunan **Materiaalinottotapa**-suodatinkentässä voit valita, näytetäänkö myös kulutus ja tuotos, joka on kirjattu automaattisesti nimikkeelle ja resurssille määritettyjen materiaalinottotapojen mukaan.</span><span class="sxs-lookup"><span data-stu-id="014aa-128">In the **Flushing Method Filter** field at the top of the journal, you can choose to also view consumption and output that is posted automatically according to the flushing methods defined for the item and resource respectively.</span></span>  

    <span data-ttu-id="014aa-129">Päiväkirjan kunkintyyppisellä rivillä näytetään vain tarvittavat kentät.</span><span class="sxs-lookup"><span data-stu-id="014aa-129">On each type of line in the journal, only the relevant fields are shown.</span></span> <span data-ttu-id="014aa-130">Loput ovat tyhjiä ja kirjoitussuojattuja.</span><span class="sxs-lookup"><span data-stu-id="014aa-130">The rest are blank and write-protected.</span></span>  

    <span data-ttu-id="014aa-131">Kun päiväkirja avataan, siihen on täytetty valmiiksi kirjattavat määrät.</span><span class="sxs-lookup"><span data-stu-id="014aa-131">When the journal is opened, it is preset with the quantities to be posted.</span></span> <span data-ttu-id="014aa-132">Jos mitään ei vielä ole kirjattu, kaikissa määräkentissä näytetään oletusarvona tuotantotilauksesta haetut oletetut määrät.</span><span class="sxs-lookup"><span data-stu-id="014aa-132">If nothing is posted so far, all quantity fields will show by default the expected quantities carried from the production order.</span></span> <span data-ttu-id="014aa-133">Jos on tehty osittaisia kirjauksia, rivien määräkentissä näytetään jäljellä olevat määrät.</span><span class="sxs-lookup"><span data-stu-id="014aa-133">If partial postings have been made, the quantity fields on the lines will show the remaining quantities.</span></span> <span data-ttu-id="014aa-134">Tilaukselle jo kirjatut määrät ja ajat näytetään päiväkirjan alaosassa todellisina tapahtumina.</span><span class="sxs-lookup"><span data-stu-id="014aa-134">The quantities and times already posted for the order are displayed at the bottom of the journal as actual entries.</span></span>  

    <span data-ttu-id="014aa-135">Voit valita, mitkä arvot **Tuotoksen määrä** -kentässä näytetään valmiina, kun päiväkirja avataan ensimmäisen kerran.</span><span class="sxs-lookup"><span data-stu-id="014aa-135">Concerning the quantities in the **Output Quantity** field, you have the option to set up which values to preset when the journal is first opened.</span></span> <span data-ttu-id="014aa-136">Valinta tehdään **Tuotannon asetukset** -sivun **Yleinen**-pikavälilehden **Esias. tuotoksen määrä** -kentässä.</span><span class="sxs-lookup"><span data-stu-id="014aa-136">This is done from the **Manufacturing Setup** page, **General** FastTab, in the **Preset Output Quantity** field.</span></span>

    > [!NOTE]  
    >  <span data-ttu-id="014aa-138">Huomaa, että vain tapahtumatyyppiä **Tuotos** oleva viimeisellä päiväkirjan rivillä oleva tuotoksen määrä muuttaa varastotasoa päiväkirjan kirjauksen yhteydessä.</span><span class="sxs-lookup"><span data-stu-id="014aa-138">Only the output quantity on the last journal line of entry type **Output** will adjust the inventory level when posting the journal.</span></span> <span data-ttu-id="014aa-139">Älä siis kirjaa päiväkirjaa, jonka viimeisellä tuotosrivillä on esiasetettu tuotoksen oletusmäärä, ennen kuin kaikki lopulliset nimikkeet on todella tuotettu.</span><span class="sxs-lookup"><span data-stu-id="014aa-139">Therefore, do not to post the journal, with the expected output quantity preset on the last output line, until all end items are actually produced.</span></span>

7.  <span data-ttu-id="014aa-142">Rekisteröi annetut määrät valitsemalla **Kirjaa**-toiminto ja sulje päiväkirja.</span><span class="sxs-lookup"><span data-stu-id="014aa-142">Choose the **Post** action to register the quantities you have entered and then close the journal.</span></span>  

<span data-ttu-id="014aa-143">Jos arvot pysyvät kirjattuina, päiväkirja sisältää nämä jäljellä olevat arvot, kun se avataan seuraavan kerran.</span><span class="sxs-lookup"><span data-stu-id="014aa-143">If values remain to be posted, the journal will contain these remaining values next time it is opened.</span></span> <span data-ttu-id="014aa-144">Kirjatut arvot näytetään päiväkirjan alaosassa todellisina arvoina.</span><span class="sxs-lookup"><span data-stu-id="014aa-144">Posted values are shown as actual values in the bottom of the journal.</span></span>  

> [!NOTE]  
>  <span data-ttu-id="014aa-145"> Jos kulutettava nimike on suljettu, päiväkirja ei kirjaa nimikkeen kulutusmääriä.</span><span class="sxs-lookup"><span data-stu-id="014aa-145">If an item that is being consumed is blocked, the journal will not post consumption quantities for that item.</span></span> <span data-ttu-id="014aa-146">Jos kuormitusryhmä tai tuotantosolu on suljettu, päiväkirja ei kirjaa kyseisen tuotosrivin tuotosmääriä eikä prosessiaikoja.</span><span class="sxs-lookup"><span data-stu-id="014aa-146">If a machine or work center is blocked, the journal will not post output quantities or process times for the output line in question.</span></span>
