# <a name="design-details-variance"></a><span data-ttu-id="24027-103">Rakennetiedot: varianssi</span><span class="sxs-lookup"><span data-stu-id="24027-103">Design Details: Variance</span></span>
<span data-ttu-id="24027-104">Varianssi on todellisten kustannusten ja vakiokustannuksen välinen ero seuraavan kaavan määrittämällä tavalla.</span><span class="sxs-lookup"><span data-stu-id="24027-104">Variance is defined as the difference between the actual cost and the standard cost, as described in the following formula.</span></span>  

 <span data-ttu-id="24027-105">todellinen kustannus – standardikustannus = varianssi</span><span class="sxs-lookup"><span data-stu-id="24027-105">actual cost – standard cost = variance</span></span>  

 <span data-ttu-id="24027-106">Jos todellinen kustannus muuttuu esimerkiksi siksi, että kirjaat nimikekustannuksen myöhemmällä päivämäärällä, tällöin varianssi päivitetään vastaavasti.</span><span class="sxs-lookup"><span data-stu-id="24027-106">If the actual cost changes, for example, because you post an item charge on a later date, then the variance is updated accordingly.</span></span>  

> [!NOTE]  
>  <span data-ttu-id="24027-107">Uudelleenarvostus ei vaikuta varianssin laskentaan, koska uudelleenarvostus vaihtaa vain varaston arvon.</span><span class="sxs-lookup"><span data-stu-id="24027-107">Revaluation does not affect the variance calculation, because revaluation only changes the inventory value.</span></span>

 <span data-ttu-id="24027-109">Seuraavassa esimerkissä kuvataan sitä, kuinka varianssi lasketaan laskutetuista nimikkeistä.</span><span class="sxs-lookup"><span data-stu-id="24027-109">The following example illustrates how variance is calculated for purchased items.</span></span> <span data-ttu-id="24027-110">Se perustuu seuraavaan skenaarioon:</span><span class="sxs-lookup"><span data-stu-id="24027-110">It is based on the following scenario:</span></span>
