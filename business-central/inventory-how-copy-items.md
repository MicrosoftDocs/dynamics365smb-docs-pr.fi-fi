<span data-ttu-id="f0fb7-104">Kun lisäät uuden nimikkeen, voit säästää aikaa kopioimalla aiemmin luodun nimikkeen **Kopioi nimike** -toiminnolla uuden nimikkeen malliksi.</span><span class="sxs-lookup"><span data-stu-id="f0fb7-104">When you add a new item, to save time, you can use the **Copy Item** function to copy an existing item to use as a template for a new item.</span></span>  

## <a name="to-copy-an-existing-item-to-a-new-item"></a><span data-ttu-id="f0fb7-105">Aiemmin luodun nimikkeen kopiointi uuteen nimikkeeseen</span><span class="sxs-lookup"><span data-stu-id="f0fb7-105">To copy an existing item to a new item</span></span>  
1. <span data-ttu-id="f0fb7-106">Valitse ![Lamppu, joka avaa Kerro, mitä haluat tehdä -toiminnon](media/ui-search/search_small.png "Kerro, mitä haluat tehdä") -kuvake, anna **Nimikkeet** ja valitse sitten liittyvä linkki.</span><span class="sxs-lookup"><span data-stu-id="f0fb7-106">Choose the ![Lightbulb that opens the Tell Me feature](media/ui-search/search_small.png "Tell me what you want to do") icon, enter **Items**, and then choose the related link.</span></span>  
2. <span data-ttu-id="f0fb7-107">Valitse kopioitava nimike ja valitse sitten **Kopioi nimike** -toiminto.</span><span class="sxs-lookup"><span data-stu-id="f0fb7-107">Select the item that you want to copy, and then choose the **Copy Item** action.</span></span>  
3. <span data-ttu-id="f0fb7-108">Täytä kentät **Kopioi nimike** -sivulla.</span><span class="sxs-lookup"><span data-stu-id="f0fb7-108">On the **Copy Item** page, fill in the fields.</span></span> [!INCLUDE[tooltip-inline-tip](includes/tooltip-inline-tip_md.md)]

    > [!NOTE]  
    > <span data-ttu-id="f0fb7-109">Voit lisätä **Kohdenimikkeen nro** -kenttään numeron, jonka haluat uuden nimikkeen hakevan.</span><span class="sxs-lookup"><span data-stu-id="f0fb7-109">You can enter a specific item number in the **Target Item No.** field that you want the new item to get.</span></span> <span data-ttu-id="f0fb7-110">Vaihtoehtoisesti voit valita aiemmin luodun numerosarjan **Kohdenimikkeen nro** -kentässä.</span><span class="sxs-lookup"><span data-stu-id="f0fb7-110">Alternatively, you can select an existing number series in the **Target No. Series** field.</span></span> <span data-ttu-id="f0fb7-111">Uusi nimike määritetään sitten seuraavalle kyseisen numerosarjan käytettävissä olevalle numerolle.</span><span class="sxs-lookup"><span data-stu-id="f0fb7-111">The new item will then be assigned the next available number in that number series.</span></span>  

5. <span data-ttu-id="f0fb7-112">Voit kopioida valintasi uuteen nimikkeeseen valitsemalla **OK**-painikkeen.</span><span class="sxs-lookup"><span data-stu-id="f0fb7-112">To copy your selections to the new item, choose the **OK** button.</span></span>
