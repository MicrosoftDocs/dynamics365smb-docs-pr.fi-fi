# <a name="create-jobs"></a><span data-ttu-id="299aa-103">Projektien luominen</span><span class="sxs-lookup"><span data-stu-id="299aa-103">Create Jobs</span></span>
<span data-ttu-id="299aa-104">Uuden projektin aloittamisen yhteydessä on luotava projektikortti sekä integroidut projektitehtävät ja projektin suunnittelurivit kahteen eri tasoon.</span><span class="sxs-lookup"><span data-stu-id="299aa-104">When you start a new project, you must create a job card with integrated job tasks and job planning lines, structured in two layers.</span></span>
