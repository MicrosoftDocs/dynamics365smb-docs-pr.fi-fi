## <a name="to-view-archived-workflow-step-instances"></a><span data-ttu-id="f0c03-109">Arkistoitujen työnkulun osavaiheen instanssien tarkasteleminen</span><span class="sxs-lookup"><span data-stu-id="f0c03-109">To view archived workflow step instances</span></span>  
1.  <span data-ttu-id="f0c03-110">Valitse ![Etsi sivu tai raportti](media/ui-search/search_small.png "Etsi sivu tai raportti -kuvake") -kuvake, kirjoita **Työnkulut** ja valitse sitten aiheeseen liittyvä linkki.</span><span class="sxs-lookup"><span data-stu-id="f0c03-110">Choose the ![Search for Page or Report](media/ui-search/search_small.png "Search for Page or Report icon") icon, enter **Workflows**, and then choose the related link.</span></span>  
2.  <span data-ttu-id="f0c03-111">Avaa työnkulku, jonka arkistoituja työnkulun osavaiheen instansseja haluat tarkastella.</span><span class="sxs-lookup"><span data-stu-id="f0c03-111">Open the workflow for which you want to view all archived workflow step instances.</span></span>  
3.  <span data-ttu-id="f0c03-112">Valitse **Työnkulku**-ikkunassa **Arkistoidut työnkulun osavaiheen ilmentymät** action.</span><span class="sxs-lookup"><span data-stu-id="f0c03-112">In the **Workflow** window, choose the **Archived Workflow Step Instances** action.</span></span>  

    <span data-ttu-id="f0c03-113">**Arkistoidut työnkulun osavaiheen ilmentymät** -ikkuna avautuu, jossa näkyvät valitun työnkulun arkistoidut työnkulun osavaiheen ilmentymät.</span><span class="sxs-lookup"><span data-stu-id="f0c03-113">The **Archived Workflow Step Instances** window opens showing the archived workflow step instances of the selected workflow.</span></span>  
4.  <span data-ttu-id="f0c03-114">Valitse vaihtoehtoisesti ![Etsi sivu tai raportti](media/ui-search/search_small.png "Etsi sivu tai raportti -kuvake") -kuvake, kirjoita **Arkistoidut työnkulun osavaiheen ilmentymät** ja valitse sitten aiheeseen liittyvä linkki.</span><span class="sxs-lookup"><span data-stu-id="f0c03-114">Alternatively, Choose the ![Search for Page or Report](media/ui-search/search_small.png "Search for Page or Report icon") icon, enter **Archived Workflow Step Instances**, and then choose the related link.</span></span>  

<span data-ttu-id="f0c03-115">**Arkistoidut työnkulun osavaiheen ilmentymät** -ikkuna avautuu, jossa näkyvät kaikki arkistoidut työnkulun osavaiheen ilmentymät.</span><span class="sxs-lookup"><span data-stu-id="f0c03-115">The **Archived Workflow Step Instances** window opens showing all archived workflow step instances.</span></span> <span data-ttu-id="f0c03-116">**Työnkulun koodi** -kentässä näet, mihin työnkulkuun arkistoitu työnkulun osavaiheen instanssi kuului.</span><span class="sxs-lookup"><span data-stu-id="f0c03-116">In the **Workflow Code** field, you can see which workflow the archived workflow step instance belonged to.</span></span>
