<span data-ttu-id="7419b-106">Kun määrität kustannuslaskennan, sinun on varmistettava, että kaikki tapahtumat on liitetty sekä kustannustyyppiin että kustannuspaikkaan ja kustannuskohteeseen.</span><span class="sxs-lookup"><span data-stu-id="7419b-106">When you set up cost accounting, you must make sure that all entries are assigned to a cost type as well as a cost center or a cost object.</span></span> <span data-ttu-id="7419b-107">Tämä tarkoittaa, että jokaisella kustannustapahtumalla täytyy olla määritettynä sekä kustannustyyppi ja joko kustannuspaikkakoodi tai kustannuskohde.</span><span class="sxs-lookup"><span data-stu-id="7419b-107">The means that each cost entry must have a cost type assigned and a cost center code or a cost object assigned.</span></span> <span data-ttu-id="7419b-108">Tämä sääntö varmistaa, että kukin kustannustapahtuma näkyy joko kustannuspaikassa tai kustannuskohteessa, mutta ei koskaan molemmissa.</span><span class="sxs-lookup"><span data-stu-id="7419b-108">This rule ensures that each cost entry appears in either the cost centers or the cost objects, but never in both places.</span></span>
