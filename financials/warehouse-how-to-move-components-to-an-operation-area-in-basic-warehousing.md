## <a name="to-move-components-to-an-operation-area-in-basic-warehouse-configurations"></a><span data-ttu-id="cb47d-123">Komponenttien siirtäminen toiminta-alueelle fyysisen varastoinnin perusmäärityksissä</span><span class="sxs-lookup"><span data-stu-id="cb47d-123">To move components to an operation area in basic warehouse configurations</span></span>

5.  <span data-ttu-id="cb47d-131">Valitse **Lähdeasiakirjat**-ikkunassa kokoonpanotilaus, jota varten haluat siirtää komponentteja, ja valitse sitten **OK**.</span><span class="sxs-lookup"><span data-stu-id="cb47d-131">In the **Source Documents** window, select the assembly order that you want to move components for, and then choose the **OK** button.</span></span>

Ikkunassa näkyvät vain varastopaikat, joissa komponentti on käytettävissä.</span><span class="sxs-lookup"><span data-stu-id="cb47d-136">If you choose the **AssistEdit** button in the **Bin Code** field on a Take line, then the **Bin Contents** window opens and only shows bins where the component is available.</span></span>  

    <span data-ttu-id="cb47d-137">Et voi muuttaa varastopaikan koodia Aseta-rivillä.</span><span class="sxs-lookup"><span data-stu-id="cb47d-137">You cannot change the bin code on a Place line.</span></span> <span data-ttu-id="cb47d-138">Vain se varastopaikan koodi hyväksytään, joka on määritetty lähdeasiakirjan komponenttirivillä.</span><span class="sxs-lookup"><span data-stu-id="cb47d-138">Only the bin code that is defined on the component line of the source document is accepted.</span></span> <span data-ttu-id="cb47d-139">Tämä tukee periaatetta, jonka mukaan rooli, joka pyytää osaa, joka on tässä toimenpiteessä kokoaja, tietää, mihin osa on sijoitettava.</span><span class="sxs-lookup"><span data-stu-id="cb47d-139">This supports the principle that the role who requests a component, which is an assembly worker in this procedure, knows where the component must be placed.</span></span> <span data-ttu-id="cb47d-140">Jos haluat asettaa komponentit toiseen varastopaikkaan, muuta ensin komponenttirivin varastopaikkakoodi ja luo sitten varastosiirtorivit uudelleen</span><span class="sxs-lookup"><span data-stu-id="cb47d-140">If you want to place the components in a different bin, then you must first change the bin code on the component line and then re-create the inventory movement lines.</span></span>

    > [!NOTE]

Movement** action.</span></span>  

    <span data-ttu-id="cb47d-147">Fyysisen varastoinnin tapahtumat luodaan kuvaten sitä, että osat ovat nyt olemassa määritetyissä varastopaikoissa kokoonpanotilausriveillä.</span><span class="sxs-lookup"><span data-stu-id="cb47d-147">Warehouse entries are created reflecting that the components now exist in the bins specified on the assembly order lines.</span></span>

    >  <span data-ttu-id="cb47d-148">Toisin kuin jos siirrät osia, joissa on varastopoiminta, kulutusta ei kirjata silloin, kun rekisteröidään varastosiirto.</span><span class="sxs-lookup"><span data-stu-id="cb47d-148">Unlike when you move components with an inventory pick, consumption is not posted when you register an inventory movement.</span></span> <span data-ttu-id="cb47d-149">Tämä vaihe on suoritettava erikseen kirjaamalla kokoonpanotilauksen tuotto ja kulutus.</span><span class="sxs-lookup"><span data-stu-id="cb47d-149">That step must be performed separately by posting the assembly order output and consumption.</span></span> <span data-ttu-id="cb47d-150">Lisätietoja on kohdassa Kokoonpanotilaus.</span><span class="sxs-lookup"><span data-stu-id="cb47d-150">For more information, see Assembly Order.</span></span>
