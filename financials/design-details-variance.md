# <a name="design-details-variance"></a><span data-ttu-id="c7afa-103">Rakennetiedot: varianssi</span><span class="sxs-lookup"><span data-stu-id="c7afa-103">Design Details: Variance</span></span>
<span data-ttu-id="c7afa-104">Varianssi on todellisten kustannusten ja vakiokustannuksen välinen ero seuraavan kaavan määrittämällä tavalla.</span><span class="sxs-lookup"><span data-stu-id="c7afa-104">Variance is defined as the difference between the actual cost and the standard cost, as described in the following formula.</span></span>  

 <span data-ttu-id="c7afa-105">todellinen kustannus – standardikustannus = varianssi</span><span class="sxs-lookup"><span data-stu-id="c7afa-105">actual cost – standard cost = variance</span></span>  

 <span data-ttu-id="c7afa-106">Jos todellinen kustannus muuttuu esimerkiksi siksi, että kirjaat nimikekustannuksen myöhemmällä päivämäärällä, tällöin varianssi päivitetään vastaavasti.</span><span class="sxs-lookup"><span data-stu-id="c7afa-106">If the actual cost changes, for example, because you post an item charge on a later date, then the variance is updated accordingly.</span></span>  

> [!NOTE]  
>  <span data-ttu-id="c7afa-107">Uudelleenarvostus ei vaikuta varianssin laskentaan, koska uudelleenarvostus vaihtaa vain varaston arvon.</span><span class="sxs-lookup"><span data-stu-id="c7afa-107">Revaluation does not affect the variance calculation, because revaluation only changes the inventory value.</span></span>

 <span data-ttu-id="c7afa-109">Seuraavassa esimerkissä kuvataan sitä, kuinka varianssi lasketaan laskutetuista nimikkeistä.</span><span class="sxs-lookup"><span data-stu-id="c7afa-109">The following example illustrates how variance is calculated for purchased items.</span></span> <span data-ttu-id="c7afa-110">Se perustuu seuraavaan skenaarioon:</span><span class="sxs-lookup"><span data-stu-id="c7afa-110">It is based on the following scenario:</span></span>

2.  <span data-ttu-id="c7afa-113">10,00 PVA hyvitetään ostovaihtelutilille.</span><span class="sxs-lookup"><span data-stu-id="c7afa-113">LCY 10.00 is credited to the purchase variance account.</span></span>

4.  <span data-ttu-id="c7afa-116">20,00 PVA veloitetaan ostovaihtelutililtä.</span><span class="sxs-lookup"><span data-stu-id="c7afa-116">LCY 20.00 is debited to the purchase variance account.</span></span> <span data-ttu-id="c7afa-117">Näin ollen netto-ostovaihteluksi tulee 10,00 PVA.</span><span class="sxs-lookup"><span data-stu-id="c7afa-117">Accordingly, the net purchase variance becomes LCY 10.00.</span></span>  
5.  <span data-ttu-id="c7afa-118">Käyttäjä määrittää nimikkeen hinnaksi 70,00 (PVA) aiemman 100,00 (PVA):n sijaan.</span><span class="sxs-lookup"><span data-stu-id="c7afa-118">The user revalues the item from LCY 100.00 to LCY 70.00.</span></span> <span data-ttu-id="c7afa-119">Tämä ei vaikuta erojen laskentaan, ainoastaan varaston arvoon.</span><span class="sxs-lookup"><span data-stu-id="c7afa-119">This does not affect the variance calculation, only the inventory value.</span></span>
