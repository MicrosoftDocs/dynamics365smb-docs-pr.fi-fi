## <a name="to-transfer-a-fixed-asset-to-a-different-department"></a><span data-ttu-id="51ad1-106">Käyttöomaisuuden siirtäminen toiselle osastolle</span><span class="sxs-lookup"><span data-stu-id="51ad1-106">To transfer a fixed asset to a different department</span></span>
<span data-ttu-id="51ad1-107">Käyttöomaisuus on ehkä siirrettävä toiselle osastolle esimerkiksi silloin, kun omaisuuserä sijoitetaan tuotanto-osastoon silloin, kun sitä vielä rakennetaan, ja kun se siirretään hallinto-osastoon valmistumisen jälkeen.</span><span class="sxs-lookup"><span data-stu-id="51ad1-107">You may need to transfer a fixed asset to a different department when, for example, you place an asset in the production department while it is under construction and then move it to the administration department when it is finished.</span></span>  

1. <span data-ttu-id="51ad1-108">Määritä uusi käyttöomaisuuserä.</span><span class="sxs-lookup"><span data-stu-id="51ad1-108">Set up a new fixed asset.</span></span> <span data-ttu-id="51ad1-109">Syötä uusi osasto **Osastokoodi**-kenttään.</span><span class="sxs-lookup"><span data-stu-id="51ad1-109">Enter the new department in the **Department Code** field.</span></span>
2. <span data-ttu-id="51ad1-110">Liitä käyttöomaisuuden poistokirja uuteen käyttöomaisuuteen.</span><span class="sxs-lookup"><span data-stu-id="51ad1-110">Assign a fixed asset depreciation book to the new fixed asset.</span></span> <span data-ttu-id="51ad1-111">Lisätietoja on kohdassa [Toimintaohje: Käyttöomaisuuden hankinta](fa-how-acquire.md).</span><span class="sxs-lookup"><span data-stu-id="51ad1-111">For more information, see [How to: Acquire Fixed Assets](fa-how-acquire.md).</span></span>
3. <span data-ttu-id="51ad1-112">Valitse ![Etsi sivu tai raportti](media/ui-search/search_small.png "Etsi sivu tai raportti -kuvake") -kuvake, syötä **KO:n uudelleenluokituspäiväkirjat** ja valitse sitten aiheeseen liittyvä linkki.</span><span class="sxs-lookup"><span data-stu-id="51ad1-112">Choose the ![Search for Page or Report](media/ui-search/search_small.png "Search for Page or Report icon") icon, enter **FA Reclass. 4. <span data-ttu-id="51ad1-113">Luo uudelleenluokittelupäiväkirjan, jonka **KO-nro**-kentässä on alkuperäinen käyttöomaisuuserä ja **Uusi KO-nro** -kentässä on uusi siirrettävä käyttöomaisuuserä.</span><span class="sxs-lookup"><span data-stu-id="51ad1-113">Create a reclassification journal where the **FA No.** field contains the original fixed asset, and the **New FA No.** field contains the new fixed asset to be moved.</span></span>

    <span data-ttu-id="51ad1-115">Rivit luodaan nyt käyttöomaisuuden KP-päiväkirjaan käyttämällä mallia ja erää, jotka olet määrittänyt **KO-päiväkirjan asetukset** -ikkunassa määritellyn poistokirjan osalta.</span><span class="sxs-lookup"><span data-stu-id="51ad1-115">Two lines are now created in the fixed asset G/L journal using the template and batch that you have specified in the **FA Journal Setup** window for the specified depreciation book.</span></span> <span data-ttu-id="51ad1-116">Lisätietoja on kohdassa [Toimintaohje: Käyttöomaisuuden poiston määrittäminen](fa-how-setup-depreciation.md).</span><span class="sxs-lookup"><span data-stu-id="51ad1-116">For more information, see [How to: Set Up Fixed Asset Depreciation](fa-how-setup-depreciation.md).</span></span>
6. <span data-ttu-id="51ad1-117">Valitse ![Etsi sivu tai raportti](media/ui-search/search_small.png "Etsi sivu tai raportti -kuvake") -kuvake, syötä **KO - KP-päiväkirjat** ja valitse sitten aiheeseen liittyvä linkki.</span><span class="sxs-lookup"><span data-stu-id="51ad1-117">Choose the ![Search for Page or Report](media/ui-search/search_small.png "Search for Page or Report icon") icon, enter **FA G/L Journals**, and then choose the related link.</span></span>    
7. <span data-ttu-id="51ad1-118">Valitse **Käyttöomaisuuden KP-päiväkirja** -ikkunassa **Kirjaa**-toiminto ja kirjaa vaiheissa 4 ja 5 suoritettu uudelleenluokittelu.</span><span class="sxs-lookup"><span data-stu-id="51ad1-118">In the **Fixed Asset G/L Journal** window, choose the **Post** action to post the reclassification that you performed in steps 4 and 5.</span></span>

<span data-ttu-id="51ad1-119">Jos yhdelle omaisuuserälle on kirjattu hankintameno, käyttöomaisuuden uudelleenluokituspäiväkirjaa voidaan käyttää jakamaan hankintameno usean omaisuuserän kesken.</span><span class="sxs-lookup"><span data-stu-id="51ad1-119">If you have posted an acquisition cost for one asset, you can use the fixed asset reclassification journal to split the acquisition cost among several assets.</span></span>
